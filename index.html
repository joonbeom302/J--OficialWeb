Welcome to Korean Programming Language, 'J++'!

🧾 소개

j++은 2025년 04월 10일에 한국에서 개발된 한글 프로그래밍 언어입니다.

이 문서에는 j++의 기본 문법, 사용 방법이 쓰여 있습니다.

📘 기본 문법

  • 변수 선언, 초기화, 재할당 처리
  • 입력
  • 출력
  • 주석
  • 다른 기능들은 추후 추가될 예정입니다. (ex: 반복문, 함수 등..)

📌 1. 변수

j++에는 세 가지 데이터 타입 [정수, 소수, 문자열]이 있습니다.

흔히 int, float, string이라고 부르죠.

변수 선언:(데이터 타입) (변수명);

변수 재할당:(변수명) = (값)

선언과 동시에 값을 넣을 수도 있습니다.

정수 a = 10;

아래는 변수 선언의 예시 입니다.

정수 a;
소수 b;
문자열 c;

정수 d = 10;
문자열 e = "안녕!"

a = 0;
b = 3.14;

📥 2. 입력

입력(입력 받을 변수명)으로 입력을 받을 수 있습니다.

다음은 입력의 예시 입니다.

정수 a;
입력(a);

a에 입력한 값이 들어갑니다.

📤 3. 출력

출력("큰 따옴표로 묶은 문자열", 변수, {원하는 만큼 쓸 수 있음})으로 출력을 할 수 있습니다.

메세지만 출력:출력("Hello, J++!");

변수만 출력:출력(변수명);

변수와 메세지 둘 다 출력:정수 a = 10; 출력("a의 값은 ", a, "입니다.");

이스케이프 시퀀스

  • 줄바꿈: \줄바꿈
  • 탭: \탭
  • 역슬래시(): \\
  • 큰따옴표("): \"
  • 작은따옴표('): \'

💬 4. 주석

주석은 총 세 가지 종류가 있습니다.

  • #: #을 쓴 부분부터 그 줄의 끝까지 주석처리

  • //: //을 쓴 부분부터 그 줄의 끝까지 주석처리

  • /* */: /*를 쓴 부분부터 */를 쓴 부분까지 모두 주석처리(줄에 영향을 받지 않음)

🚀 사용 방법

  1. j++ 파일을 실행시킬 폴더를 만든다.
  2. 폴더 속에 j++ 인터프리터 코드를 넣을 파이썬 파일을 생성한다.
  3. j++파일을 생성한다.
  4. 터미널에 아래 코드를 입력한다.python (인터프리터).py (j++파일).j++
